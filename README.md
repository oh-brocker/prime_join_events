## 🚀 Про плагін

**PrimeJoinEvents** — це легкий, розширюваний плагін для Minecraft-серверів, який дозволяє **гнучко керувати подіями входу гравців** без зайвої складності.

---

## ✨ Основні можливості

- 💬 Кастомні повідомлення в чаті
- 🏷️ Title та Subtitle при вході
- 🔊 Відтворення звуків через Adventure API
- 👤 Окрема логіка для:
  - першого входу гравця
  - повторних входів
- 🎨 Підтримка MiniMessage (`<gradient>`, `<bold>`, тощо)

---

## 🧠 Філософія

PrimeJoinEvents **не намагається бути всім одразу**. Плагін писався під потреби реального серверу. Усі рішення зроблені **свідомо** і легко змінюються під конкретний сервер.
